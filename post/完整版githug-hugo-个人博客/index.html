🎉 Windown 本地安装 Hugo#

打开 Hugo windows installation#

使用管理员权限 winget 包管理工具安装#

winget install Hugo.Hugo.Extended

打开 Git windows installation#

winget install --id Git.Git -e --source winget

Hugo 建立新项目#

hugo new site web3blog

cd 进入项目根目录#

cd web3blog

安装 hugo 模板#

git init
git clone https://github.com/adityatelange/hugo-PaperMod themes/PaperMod --depth=1
git submodule add --depth=1 https://github.com/adityatelange/hugo-PaperMod.git themes/PaperMod

修改 hugo.toml 配置文件#

baseURL = 'https://「你的Github账号」.github.io/web3blog/'
languageCode = 'en-us'
title = 'Web3 撸毛日记'
theme = 'PaperMod'

enableEmoji = true # 开启MarkDown表情包解析

修改文章模版 achetypes/default.md#

---
title: "{{ replace .Name "-" " " | title }}"
date: {{ .Date }}
draft: false
ShowToc: true # 开启 Table Of Content 功能
TocOpen: true # 展开 Table Of Content
---

发布第一篇文章#

hugo new post/第一篇博文.md

🎉 Github pages 绑定 Vscode#

配置Github SSH链接#

ssh-keygen -t ed25519 -C "your_email@example.com"

生成密钥后看提示找到文件 id_ed25519.pub

使用 cat ~/.ssh/id_ed25519.pub 命令读取文件内容填入Github

点击创建 githug 新项目#

  • 使用SSH链接Github
echo "# web3-blog" >> README.md
git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin git@github.com:davidpythonseo/web3-blog.git
git push -u origin main
  • Githug 创建分支 gh-pages

  • 设置分支读写权限

  • 写入GitHub Actions脚本
mkdir -p .github/workflows
touch .github/workflows/deploy.yml
vim .github/workflows/deploy.yml
  • 填入脚本代码
name: Publish to GH Pages
on:
  push:
    branches:
      - main
  pull_request:

jobs:
  deploy:
    runs-on: ubuntu-latest
    steps:
      - name: Checkout source
        uses: actions/checkout@v3
        with:
          submodules: true

      - name: Checkout destination
        uses: actions/checkout@v3
        if: github.ref == 'refs/heads/main'
        with:
          ref: gh-pages
          path: built-site

      - name: Setup Hugo
        run: |
          curl -L -o /tmp/hugo.tar.gz 'https://github.com/gohugoio/hugo/releases/download/v0.110.0/hugo_extended_0.110.0_linux-amd64.tar.gz'
          tar -C ${RUNNER_TEMP} -zxvf /tmp/hugo.tar.gz hugo                    
      - name: Build
        run: ${RUNNER_TEMP}/hugo

      - name: Deploy
        if: github.ref == 'refs/heads/main'
        run: |
          cp -R public/* ${GITHUB_WORKSPACE}/built-site/
          cd ${GITHUB_WORKSPACE}/built-site
          git add .
          git config user.name 'dhij'
          git config user.email 'davidhwang.ij@gmail.com'
          git commit -m 'Updated site'
          git push
  • 三件套命令
git add .
git commit -m "更新"
git push
